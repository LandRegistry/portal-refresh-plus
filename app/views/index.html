  1. Index page - Additional Group filter, Create new group button and group column in data table
  2. Create new group - This page allows users to create groups, add team members and applications
  3. Index page updated - This page contains additional applications with a new custom group filter added
  4. View group - This is a 'read-only' page to view an existing group
  5. Edit group - This is a page allows user to edit an existing group
