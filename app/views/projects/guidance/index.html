Guidance page

Overview

The aim of this project is enable users to create groups and projects. Applications could be associated with a group or project, enabling all members also associated with the group or project, to be able to view them intuitively (rather than searching for them individually).

Project team

  • Andy Carr - Interaction design
  • Lauren Dowling - Interaction design
  • James Thomas - Content design
  • Jon Parker - User research
  • Juan Pinto Del Rio - User research
