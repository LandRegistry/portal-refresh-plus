View applications

View applications allows you to search for applications from your business unit, submitted via the HM Land Registry Portal or saved within the Digital Registration Service (DRS). Applications are displayed when saved, live, and for 90 calendar days following their completion or cancellation. This guide helps to explain what the features on the page do.

Filter

Filter by requisition

Use this to display applications that have a requisition(s) listed in the ‘Review’ correspondence category. You can filter by:

  • Review - These applications have outstanding requisition points raised against them.
  • Warning of cancellation - These applications have had a Warning of cancellation letter issued against them relating to outstanding requisition points.

Note:

If a response has already been provided from your business unit, HMLR will need to confirm all of the requisition points have been satisfactorily replied to. Once this has been established, your application can be progressed and will no longer show in the results for this filter.

Filter by application type

Use this to display applications by their type. You can filter by:

  • Register update (an application that changes an existing registered title)
  • Register create (an application to make a new registered title)
  • Register query (an enquiry or search regarding information held at HM Land Registry)

Filter by application progress

Use this to display applications by their current progress status. You can filter by:

  • Cancelled
  • Completed
  • Delayed
  • Received
  • Saved

Note:

Delayed results include applications with a progress status of:

  • Delayed
  • Delayed Prior Pending Application(s) or Official Search(es)
  • Delayed - Extension of Time Granted

Received results include applications with a progress status of:

  • Received: Priority Protected
  • Received: Priority Protected – Awaiting Processing
  • Received: Priority Protected – Set-up activity underway
  • In progress

Applications that are displayed without a progress status will also be shown within the results of the Received filter option.

Saved results show pre-submitted applications that have been stored by you only, in the Digital Registration Service. Currently, your business unit colleagues’ ‘Saved’ application are not displayed by the service.

Search

Search will locate applications from within your business unit. Only your personally submitted applications will be returned if your business unit has decided to opt out of the view colleagues’ applications functionality.

You can search for applications using:

  • Title number or HMLR reference
  • Customer reference
  • Colleague

For Search by either Title number or HMLR reference, or Customer reference, the results will show applications that contain the characters (a minimum of 3 is required) entered as the search term. For Search by Colleague, the number of names will reduce as more characters are entered. Once a name is selected from the list, that colleague’s applications will be displayed.

Note:

The search field does not recognise some special characters (e.g. ç, à, è, ä, û). When searching for a colleague’s name, please use standard alphabet characters if possible.

Reset search and filters

Allows you to clear any existing search term, sort or filter. When the page is reset the system returns to displaying only the applications submitted by you via the portal.

Export Applications

Allows you to extract information relating to all or a selected sample of applications based on the search and filter options you have used.

Note:

Currently, the Export Applications function is not available for users of the Internet Explorer (IE) browser.

Sort

Use sort to order your list of applications. You can sort by:

  • Title number
  • Customer reference
  • Received by HMLR

Initial Application Information (collapsed row)

Warning symbol

A warning symbol is displayed when an application has a requisition listed in the ‘Review’ correspondence category.

Title number

Shows the title number for the application (if available).

Note 1:

For multi title applications with a progress status of ‘Saved’, only the lowest title number is displayed.

Note 2:

For Land Charges Registration View and Land Charges Office Copy applications it shows the Land Charges Registration Number.

Customer reference

This is the reference provided when the application was submitted.

Application type

The type of application lodged (for example, Transfer of Part, Official Search of Whole with Priority).

Received by HMLR

This is the date the application was received by HM Land Registry.

Application progress

The high-level progress status of the application (for example, Cancelled, Completed, Delayed, Received, Saved). This status is only displayed for applications that update or create a Register.

Note:

Received results include applications with a progress status of:

  • Received: Priority Protected
  • Received: Priority Protected – Awaiting Processing
  • Received: Priority Protected – Set-up activity underway
  • In progress

Applications that are displayed without a progress status will also be shown within the results of the Received filter option. Delayed results include applications with a progress status of:

  • Delayed
  • Delayed Prior Pending Application(s) or Official Search(es)
  • Delayed - Extension of Time Granted

Estimated completion date (ECD)

Is a date by which the application is likely to be completed. It is not based on the individual application, but is an estimate of when the majority (90%) of that application type will be completed.

ECD key points

  • displayed for Register Create (e.g. Transfers of Part and New Leases) and Register Update applications
  • distinguishes between developer and non-developer Register Create application types
  • calculated for the application type and not the individual application
  • calculated using the time in which 90% of that application type was completed over the previous 30 calendar days
  • may take up to an hour to be displayed
  • calculated on and from the day an application is received by HMLR
  • updated every 30 calendar days until the application is completed or cancelled
  • updated if HMLR correspondence is issued (first item of correspondence only)
  • expedited applications will be prioritised but the ECD displayed will not be updated to reflect this

Where there is no date, one of the following will be displayed:

  • Calculating
    This is shown when the ECD for the application is being calculated for the first time.
  • Exceeded
    This is shown when the ECD for the application has elapsed - 10% of applications fall into this category.
  • Unavailable
    This is shown when an error has occurred, preventing an ECD from being displayed. When available, the date will then appear.
  • Not applicable
    This is shown for all Register Query application types and applications that have a progress status of ‘Saved’. HMLR does not provide an ECD for this type of application.

Note:

On completion or cancellation of an application, any ECD information is removed, leaving the column entry blank.

Show / hide details (+)

Expands the row so more information about the application can be viewed.

Additional Application Information (expanded row)

Application progress - details

This shows the progress of an application by the following stages:

  • Received: Priority Protected
    The application has been received and added to our Day List. Its priority has been protected, but no processing activity has taken place.
  • Received: Priority Protected – Awaiting Processing
    This is shown when an application is being held on an office’s backlog list to await processing. No processing activity has yet taken place. The application will normally be dealt with in accordance with its priority order.
  • Received: Priority Protected – Set-up activity underway
    This is shown when the parent title is a developer title and there are setting up activities that need to be completed before applications can proceed.
  • In Progress
    Processing activities are taking place. There are no known events or activities causing the application to be delayed at this time.
  • Delayed Prior Pending Applications(s) or Official Search(es)
    The application has prior pending application(s) or official search(es) with priority on our Day List.
  • Delayed
    Events or activities are causing progress of the application to be delayed.
  • Delayed - Extension of Time Granted
    This is shown when a request for additional time to reply to one or more requisition points has been approved. The amount of additional time allowed can vary and is at the discretion of the Chief Land Registrar based upon the individual circumstances. Subject to approval, a minimum of 10 working days is normally granted in the first instance, while subsequent requests will be treated on their merits. Requests for extension of time can be submitted via the ‘Still Need To Contact Us’ link located within the Application Enquiry service

HMLR reference

The unique reference HM Land Registry allocates to an application.

Priority period

Shown for Official Search with Priority applications only. The full period during which any application protected by a search retains its priority.

Parent title(s)

This is the existing title or titles out of which a Transfer of Part or New Lease application has been granted.

Correspondence

Shows notices, requisitions and some letters issued by HM Land Registry connected with an application under the categories:

  • Review
    These items raise points that are outstanding against the application. Provide a full response to allow the application to proceed. If a response has already been provided, HMLR will need to confirm all points have been satisfactorily replied to before progressing the application.
  • Resolved
    Once HMLR has confirmed all the ‘Review’ points have been satisfactorily replied to, the correspondence item will move from the ‘Review’ section to this category.
  • Information only
    These items are for information only and do not need a response.

Correspondence can include:

  • Letters - Letter issued on [date]
    This is shown when HM Land Registry issues a letter in respect of the application.
  • Notice (Objection):
    • Notice(s) issued on [date] Expires on [date]
      This is shown when HM Land Registry issues a Notice in respect of the application. HM Land Registry serves notices to interested parties to help prevent wrongful registration. A copy notice is not sent to the applicant. Notices inform the recipient that HM Land Registry has received an application, or is otherwise proposing an alteration that may adversely affect their interest. These allow a time period for objection, but also invite the addressee to consent to the application in order to shorten the period of delay.
  • Notice (Information):
    • Notice(s) issued on [date] Expires on [None]
      This is shown when HM Land Registry issues a Notice for information purposes only. This type of Notice does not have an expiry date and does not require a response.
    • Requisition(s) issued on [date]
      This is shown when HM Land Registry issues a requisition asking for additional information in support of the application. A requisition will not be deemed to have been satisfied until all outstanding points raised have been fully replied to.
    • Warning of Cancellation issued on [date]
      This is shown when HM Land Registry issues a warning that the application will be cancelled unless a full response to a requisition is received.
    • Letter of Cancellation issued on [date] This is shown when HM Land Registry have cancelled the application.

Files

Shows the documentation electronically issued by HM Land Registry to the applicant on completion of the application. These can be viewed and downloaded.

Show / hide details (-)

Collapses the row so less information about your application is displayed.

Actions

Respond

HMLR correspondence does not need to have been received on an application to use ‘Respond’.

Respond allows you to:

  • reply to a requisition or other HMLR correspondence
  • add documents and notes to the application
  • request an extension of time in reply to HMLR correspondence
Note:
This feature does not support applications lodged via Network Services.

Application Enquiry

This gives access to additional information held about the application on the HMLR Day List for register create, register update and official search (with priority) applications.

Application Enquiry allows you to:

  • request an update on the application
  • apply to expedite the application
  • view the prior pending applications that affect the application
  • view discharge of a charge activity

Save as PDF

Allows you to save a copy of the application details displayed.

Note:
Currently, the Save as PDF function is not available for users of the Internet Explorer (IE) browser.
