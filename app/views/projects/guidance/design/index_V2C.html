Guidance

View applications

View applications allows you to search for applications from your business unit, submitted via HM Land Registry portal or saved within the Digital Registration Service (DRS). Applications are displayed when saved, live, and for 90 calendar days following their completion or cancellation. The purpose of this guide is to explain what each feature in view applications does.

Filter applications

Filter by requisition

Display applications that have a requisition listed in the ‘Review’ correspondence category. Applications can be filtered by:

  • Review - These applications have outstanding requisition points raised against them
  • Warning of cancellation - These applications have had a warning of cancellation letter issued against them relating to outstanding requisition points
Note: If a response has already been provided from your business unit, HM Land Registry will need to confirm all of the requisition points have been satisfactorily replied to. Once this has been established, an application can be progressed and will no longer show in the results for this filter.

Filter by application type

Display applications by their type. Filter by:

  • Register update (an application that changes an existing registered title)
  • Register create (an application to make a new registered title)
  • Register query (an enquiry or search regarding information held at HM Land Registry)

Filter by application progress

Display applications by their current progress status. Filter by:

  • Cancelled
  • Completed
  • Delayed
  • Received
  • Saved

Delayed results include applications with a progress status of:

  • Delayed
  • Delayed Prior Pending Application(s) or Official Search(es)
  • Delayed - Extension of Time Granted

Received results include applications with a progress status of:

  • Received: Priority Protected
  • Received: Priority Protected – Awaiting Processing
  • Received: Priority Protected – Set-up activity underway
  • In progress

Applications that are displayed without a progress status will also be shown within the results of the received filter option.

Saved results show pre-submitted applications that have been stored by you and your business unit colleagues in the Digital Registration Service.

Search for applications

Search for applications will locate applications from within your business unit. Only personally submitted applications will be returned if your business unit has decided to opt out of the view colleagues’ applications functionality.

Find applications by title number, HM Land Registry reference, customer reference and colleague.

To search by title number, HM Land Registry reference or customer reference, the results will show applications that contain the characters (a minimum of 3 is required) entered as the search term. For search by colleague, the number of names will reduce as more characters are entered. Once a name is selected from the list, that colleagues’ applications will be displayed.

Note: The search field does not recognise some special characters, for example, ç, à, è, ä, û. When searching for a colleague’s name, please use standard alphabet characters if possible.

If a colleague only has saved applications present, their name will not appear on the list of users displayed during a colleague search. However, these applications can be located using either a title number or a customer reference search.

Reset search and filters

Allows you to clear any existing search term, sort, or filter. When the page is reset, the system returns to displaying only the applications submitted by you.

Exporting applications

Extract information relating to all, or a selected sample of applications based on the search and filter options you have used.

Note: Not available on the Internet Explorer browser.

Sorting applications

Use to order a list of applications by title number, customer reference number or received by HM Land Registry.

View application information within the collapsed row

Warning symbol

A warning symbol is displayed when an application has a requisition listed in the ‘Review’ correspondence category.

Title number

Shows the title number for the application (if available).

Note: For multi-title applications with a progress status of ‘Saved’, only the title number inputted first into the Digital Registration Service is displayed.

For Land Charges Registration View and Land Charges Office Copy applications, it shows the Land Charges Registration Number.

Customer reference

This is the reference provided when the application was submitted.

Application type

The type of application lodged. For example, a ‘Transfer of Part’ or an ‘Official Search of Whole with Priority’.

Received by HMLR

This is the date the application was received by HM Land Registry.

Application progress

The high-level progress status of the application (for example: Cancelled, Completed, Delayed, Received, Saved). This status is only displayed for applications that update or create a Register.

Received results include applications with a progress status of:

  • Received: Priority Protected
  • Received: Priority Protected – Awaiting Processing
  • Received: Priority Protected – Set-up activity underway
  • In progress

Applications that are displayed without a progress status will also be shown within the results of the ‘Received’ filter option. ‘Delayed’ results include applications with a progress status of:

  • Delayed
  • Delayed Prior Pending Application(s) or Official Search(es)
  • Delayed - Extension of Time Granted

Estimated completion date

This is the date by which the application is likely to be completed and is not based on the individual application. It is an estimate of when the majority (90%) of that application type will be completed. ECD:

  • is displayed for ‘Register Update’ and ‘Register Create’ applications. For example, ‘Transfers of Part’ and ‘New Leases’
  • distinguishes between developer and non-developer ‘Register Create’ application types
  • is calculated for the application type and not the individual application
  • is calculated using the time in which 90% of that application type was completed over the previous 30 calendar days
  • may take up to an hour to be displayed
  • is calculated on and from the day an application is received by HM Land Registry
  • updated every 30 calendar days until the application is completed or cancelled
  • updated if HM Land Registry correspondence is issued (first item of correspondence only)
  • expedited applications will be prioritised but the ECD displayed will not be updated to reflect this

Where there is no date, one of the following will be displayed.

Calculating

This is shown when the ECD for the application is calculated for the first time.

Exceeded

This is shown when the ECD for the application has elapsed - 10% of applications fall into this category.

Unavailable

This is shown when an error has occurred, preventing an ECD from being displayed. When available, the date will then appear.

Not applicable

This is shown for all ‘Register Query’ application types and applications that have a progress status of ‘Saved’. HM Land Registry does not provide an ECD for these types of applications.

Note: On completion or cancellation of an application, any ECD information is removed, leaving the column entry blank.

Show/hide details (+)

Expands the row so more information about the application can be viewed.

View application progress within the expanded row

This shows the progress of an application through the following stages.

Received: Priority Protected

The application has been received and added to HM Land Registry Daylist. The priority has been protected, but no processing activity has taken place.

Received: Priority Protected – Awaiting Processing

This is shown when an application is being held on an office’s backlog list to await processing. No processing activity has yet taken place. The application will normally be dealt with by its priority order.

Received: Priority Protected – Set-up activity underway

This is shown when the parent title is a developer title and has set up activities to complete before applications can proceed.

In Progress

Processing activities are taking place. No known events or activities are causing the application to be delayed at this time.

Delayed Prior Pending Application(s) or Official Search(es)

The application has prior pending application(s) or official search(es) with priority on the Daylist.

Delayed

Events or activities are causing the progress of the application to be delayed.

Delayed - Extension of Time Granted

This is shown when a request for additional time to reply to one or more requisition points has been approved. The amount of additional time allowed can vary and is at the discretion of the Chief Land Registrar based on individual circumstances. Subject to approval, a minimum of 10 working days is normally granted in the first instance, while subsequent requests will be treated on their merits. Requests for an extension of time can be submitted via the ‘Still Need To Contact Us’ link located within the ‘Application Enquiry’ service.

HMLR reference

The unique reference HM Land Registry allocates to an application.

Priority period

Shown for ‘Official Search with Priority’ applications only. The full period during which any application protected by a search retains its priority.

Parent title(s)

This is the existing title or titles out of which a ‘Transfer of Part’ or ‘New Lease’ application has been granted.

View correspondence

Shows notices, requisitions and some letters issued by HM Land Registry connected with an application under these categories.

Review

These items raise points that are outstanding against the application. Provide a full response to allow the application to proceed. If a response has already been provided, HM Land Registry will need to confirm all points have been satisfactorily replied to before progressing with the application.

Resolved

Once HM Land Registry has confirmed all the review points have been satisfactorily replied to, the correspondence item will move from the ‘Review’ section to the ‘Resolved’ section.

Information only

These items are for information only and do not need a response.

Note: Items under this category may include correspondence sent to parties outside of your organisation. You will not be able to view the content of these items.

Correspondence can also include these categories.

Letter issued on [date]

This is shown when HM Land Registry issues a letter in respect of the application.

Notice (Objection):
Notice(s) issued on [date] Expires on [date]

This is shown when HM Land Registry issues a notice in respect of the application. HM Land Registry serves notices to interested parties to help prevent wrongful registration. A copy notice is not sent to the applicant. Notices inform the recipient that HM Land Registry has received an application or is otherwise proposing an alteration that may adversely affect their interest. These allow for a period of time for objection but also invite the addressee to consent to the application to shorten the period of delay.

Notice (Information):
Notice(s) issued on [date] Expires on [None]

This is shown when HM Land Registry issues a notice for information purposes only. This type of notice does not have an expiry date and does not require a response.

Requisition(s) issued on [date]

This is shown when HM Land Registry issues a requisition asking for additional information in support of the application. A requisition will not be deemed to have been satisfied until all outstanding points raised have been fully replied to.

Warning of cancellation issued on [date]

This is shown when HM Land Registry issues a warning that the application will be cancelled unless a full response to a requisition is received.

Letter of cancellation issued on [date]

This is shown when HM Land Registry cancels the application.

View files and take actions

Files

Shows the documentation electronically issued by HM Land Registry to the applicant on completion of the application. These can be viewed and downloaded.

Respond

HM Land Registry correspondence does not need to have been received on an application to use ‘Respond’.

Respond allows you to:

  • reply to a requisition or other HM Land Registry correspondence
  • add documents and notes to the application
  • request an extension of time in reply to HM Land Registry correspondence
Note: This feature does not support applications lodged via Network Services.

Application Enquiry

Provides access to additional information held about the application on the HM Land Registry Daylist for 'Register Create’, 'Register Update’ and ‘Official Search (with Priority)’ applications.

Application Enquiry allows you to:

  • request an update on the application
  • apply to expedite the application
  • view the prior pending applications that affect the applications
  • view discharge of a charge activity

Save as PDF

Save a copy of the application details displayed.

Note: Not available on the Internet Explorer browser.

Open application

Access pre-submitted applications that have been saved by you only, in Digital Registration Service.

Note: This functionality is not available for pre-submitted applications that have been saved by your business unit colleagues.

Show/hide details (-)

Collapses the row so less information about your application is displayed.
