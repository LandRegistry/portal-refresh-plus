Homepage

Overview

The aim of this project is to create a modern feeling homepage that is functional, accessible and meets our user's needs. The current homepage is showing its age and is primarily used as a gateway through to the service the user requires.

Project team

  • Andy Carr - Interaction design
  • James Thomas - Content design
  • Jon Parker - User research
  • Juan Pinto Del Rio - User research
